使用vue3.0 + element-plus做的后台系统

项目线上地址：http://wyz.baicaitang.top/#/login

=======

学习使用到：

- vue3.0
- element-plus
- vuex
- Axios拦截器
- 检测toKen
- cookie存储，sessionStorage存储、localStorage存储
- svg 图片拼接成 雪碧图
- 组件化开发
- 动态路由
- 路由守卫
- 按钮权限
- 开发环境跨域，线上跨域

# vue-admin

## 安装依赖
```
yarn install
```

### 启动服务
```
yarn serve
```

### 打包
```
yarn build
```
